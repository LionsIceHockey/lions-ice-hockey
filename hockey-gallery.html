📸 How to add your photos

This page is ready to host. To add real photos, simply edit the HTML:

  1. Open hockey-gallery.html in any text editor (Notepad, VS Code, etc.)
  2. Find the PHOTOS array near the top of the <script>
  3. Replace the placeholder entries with your image filenames and captions
  4. Put your photo files in the same folder as the HTML file
  5. Host the whole folder for free on GitHub Pages or Netlify Drop
  6. Generate a QR code pointing to your site URL at qr.io or goqr.me
